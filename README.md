# Automobilio variklio elektroninės valdymo sistemos simuliatorius

## Įvadas

Ši programa abstrakčiai įgyvendina automobilio variklio elektroninę valdymo sistemą ir simuliuoja šios sistemos veikimą, saugant duomenis kaip RPM, temperatūra, purškiamas degalų kiekis, degalų sanaudos ir t.t. Vartotojas paleidęs programą gali nustatyti variklio parametrus tam tikruose rėžiuose, tada mato grafinę sąsają, kurioje gali kelti ir mažinti variklio apsukas ir matyti kaip keičiasi variklio darbo parametrai.

### Naudojimas

Programa iš pradžių paklaus vartotojo įvesti variklio duomenis, tada matys grafinę sąsają, kur gali kelti ir mažinti variklio apsukas su klavišais 'Arrow Up' ir 'Arrow Down'
